# ChocoCrunch Analytics

## Project Overview
ChocoCrunch Analytics is a Python-based data analytics project that analyzes chocolate products from the OpenFoodFacts API.  
It cleans, transforms, and stores data in MySQL, then visualizes insights through an interactive Streamlit Dashboard.

## Steps to Run

1. Run `backend.py`  
   - This file extracts, cleans, and prepares chocolate product data.

2. Run `streamlit_dashboard.py`  
   - This file opens the interactive dashboard in your browser at http://localhost:8501

## Tools Used
- Python (for data extraction and processing)
- MySQL (for database storage)
- Streamlit (for dashboard visualization)
- Pandas, Plotly, Matplotlib (for analysis and graphs)
